Type GitHub Code GitHub Code functionalities
CONFIGURE TOOLING $ git config --global user.name "[name]" Sets the name you want atached to your commit transactions
CONFIGURE TOOLING $ git config --global user.email "[email address]" Sets the email you want atached to your commit transactions
CONFIGURE TOOLING $ git config --global color.ui auto Enables helpful colorization of command line output
CREATE REPOSITORIES $ git init [project-name] Creates a new local repository with the specified name
CREATE REPOSITORIES $ git clone [url] Downloads a project and its entire version history
MAKE CHANGES $ git status Lists all new or modified files to be commited
MAKE CHANGES $ git diff Shows file differences not yet staged
MAKE CHANGES $ git add [file] Snapshots the file in preparation for versioning
MAKE CHANGES $ git diff --staged Shows file differences between staging and the last file versions
MAKE CHANGES $ git reset [file] Unstages the file, but preserve its contents
MAKE CHANGES $ git commit -m "[descriptive message]" Records file snapshots permanently in version history
REFACTOR FILENAMES $ git rm [file] Deletes the file from the working directory and stages the deletion
REFACTOR FILENAMES $ git rm --cached [file] Removes the file from version control but preserves the file locally
REFACTOR FILENAMES $ git mv [file-original] [file-renamed] Changes the file name and prepares it for commit
